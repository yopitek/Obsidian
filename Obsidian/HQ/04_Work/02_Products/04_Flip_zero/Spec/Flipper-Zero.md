# Flipper Zero
## Portable Multi-Tool for Pentesters and Hackers

### Overview
A pocket-sized multi-tool for pentesters and geeks in a toy-like body. Flipper Zero is capable of interacting with digital systems in real life — it can read, copy, and emulate RFID and NFC tags, radio remotes, iButton keys, infrared signals, and much more. Fully open-source and community-driven with continuous firmware updates.

---

### Technical Specifications

| Parameter | Value |
|---|---|
| Display | 1.4" monochrome LCD, 128×64 px |
| CPU | STM32WB55 (ARM Cortex-M4 @ 64 MHz + Cortex-M0+ @ 32 MHz) |
| Sub-GHz Radio | CC1101 transceiver — 300–928 MHz |
| NFC | ST25R3916 — ISO 14443A/B, ISO 15693, FeliCa |
| RFID (125 kHz) | EM4100, HID Prox, Indala, Pyramid, Paradox |
| Infrared | TX/RX — 850 nm, range up to 10 m |
| iButton | DS1990A, Cyfral, Metakom |
| Bluetooth | BLE 5.0 (Cortex-M0+ co-processor) |
| USB Interface | USB-C (CDC serial + firmware update + charging) |
| Battery | 2000 mAh Li-Po, ~30 days standby, USB-C charge |
| Storage | microSD slot (16–32 GB recommended, not included) |
| GPIO Header | 18-pin — 3.3 V, SPI, I²C, UART, ADC, PWM |
| Dimensions | 100 × 40 × 26 mm |
| Weight | ~102 g |
| Firmware | Open-source (MIT) — updated via app or desktop |

### Wireless Capabilities

| Protocol | Frequency | Notes |
|---|---|---|
| Sub-GHz | 300–928 MHz | Read/write/emulate — garage doors, gate remotes, weather stations |
| NFC | 13.56 MHz | ISO 14443A/B, FeliCa — credit cards, access badges, NFC tags |
| RFID | 125 kHz | EM4100, HID Prox, Indala — old-school keycards |
| Infrared | 850 nm IR | TV/AC remotes — TX and RX capable |
| iButton | 1-Wire | DS1990A, Cyfral, Metakom — apartment intercom keys |
| Bluetooth LE | 2.4 GHz | BLE 5.0 — mobile app control, keyboard/mouse emulation |
