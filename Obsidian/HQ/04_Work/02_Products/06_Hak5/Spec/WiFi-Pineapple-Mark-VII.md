# WiFi Pineapple Mark VII

> The industry standard WiFi pentest platform has evolved. Equip your red team with the WiFi Pineapple® Mark VII. Newly refined. Enterprise ready.

## Overview

The WiFi Pineapple Mark VII is a portable WiFi auditing platform used by pentesters and red teams worldwide. It features the patented PineAP suite for rogue access point attacks, passive/active recon, and automated campaign reporting.

## Key Features

- **PineAP Suite** — Rogue AP attacks for Open, WPA-PSK, and WPA-Enterprise networks
- **Recon** — Real-time visualization of the wireless landscape
- **Campaigns** — Automated pentests with actionable intelligence reports
- **Cloud C² Integration** — Remote access and management via Cloud C²
- **Modules** — Extensible with community and first-party modules
- **Web Shell** — Built-in terminal access from the browser
