# Plunder Bug

> Pocket-sized LAN Tap with USB-C convenience. Bug Ethernet connections on the go.

## Overview

The Plunder Bug by Hak5 is a pocket-sized LAN Tap that lets you intercept ("bug") Ethernet connections with USB-C convenience. It sits inline on an Ethernet link and presents a capture interface to your computer for passive or active monitoring.

## Key Features

- **Inline LAN Tap** — Sits between two Ethernet devices; invisible to the network
- **USB-C Interface** — Connect to your laptop for packet capture
- **Passive & Active Modes** — Switch between monitoring-only and full inline bridge
- **Wireshark Ready** — Captured traffic viewable in standard tools
- **Can Work as Simple Switch** — Combine with Shark Jack for flexible setups

## Operating Modes

| Mode | Description |
|------|-------------|
| **Passive** | Monitor-only tap; captures traffic without affecting link |
| **Active** | Full inline bridge; enables MITM capabilities |

## Use Cases

- Passively sniff Ethernet traffic during engagements
- Capture credentials and sessions from wired connections
- Network troubleshooting and protocol analysis
- Combine with Shark Jack as a simple Ethernet switch
- Feed captures into Wireshark for analysis

## Getting Started

1. Plug Plunder Bug inline on the Ethernet link (between device and switch/router)
2. Connect USB-C to laptop
3. Install drivers if required
4. Open Wireshark / tcpdump on the capture interface
5. Traffic flows through; captures on your machine
