# LAN Turtle

> Drop a LAN Turtle. Get a Shell.

## Overview

The LAN Turtle is a covert systems administration and penetration testing tool disguised as a generic USB Ethernet adapter. It provides stealth remote access, network intelligence gathering, and man-in-the-middle surveillance capabilities through a simple graphic shell.

## Overview

Housed in a generic "USB Ethernet Adapter" case, the LAN Turtle blends into IT environments naturally. Plug it inline between a computer and the network to gain persistent access to the entire LAN.

## Key Features

- **Man-in-the-Middle** — Intercept and capture all interesting network data
- **MicroSD Storage** — Save network traffic captures; load favorite pentest tools
- **Module System** — Easy setup with repository browser; modules via checkbox UI
- **Simple Module API** — Write custom modules quickly
- **Cloud C² Ready** — Pentest from anywhere; exfiltrate loot; drop into root shell via web browser
- **Covert Appearance** — Looks like an innocent USB Ethernet adapter

## Pentester Use Cases

- Pivot with persistent Meterpreter session (Metasploit)
- Network scanning with nmap
- DNS spoofing to phishing sites
- Data exfiltration via C2EXFIL or SSHFS
- Man-in-the-Middle inline capture of browser traffic
