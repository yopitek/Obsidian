# WiFi Pineapple Pager

> Taking the industry standard in WiFi pentesting tools mobile, with 1990s flair.

## Overview

The WiFi Pineapple Pager is the 8th generation of the WiFi Pineapple platform reimagined as a fully standalone, portable device. It features the rebuilt PineAP v8 engine (100× faster), a built-in display, battery, and the first DuckyScript™-powered payload system on a WiFi Pineapple. No laptop required.

## Key Features

- **PineAP v8** — 8th generation engine, 100× faster, rebuilt from the kernel up
- **Payload System** — First WiFi Pineapple to run DuckyScript™ + Bash payloads
- **Alert Payloads** — Fire alerts (vibrations, ringtones, visuals) on WiFi events
- **Standalone Operation** — Built-in display, controls, and battery
- **Virtual Pager Interface** — Web UI to mirror screen and press buttons remotely
- **USB-C Ethernet** — Integrated USB Ethernet for wired access
- **Root Linux Access** — SSH, web terminal, Bash, Python
- **Hardware Mods** — Extensible via standard USB interface (GPS, Ethernet, Mesh)
- **Fully Hackable** — Custom themes, ringtones, hardware mods

## Attacks Supported

- Rogue AP (Open, WPA-PSK)
- Man-in-the-Middle
- Deauth
- Handshake capture
- OSINT collection
- Event-driven alert payloads

## Scripting

- DuckyScript™ (supported in Payload Studio v1.5 Community & Pro)
- Bash
- Python (via Linux base)
