# Cloud C² (Cloud C2)

> Remote pentesting made easy. Deploy Hak5 gear, ship it to the client, access it from anywhere.

## Overview

Cloud C² is Hak5's self-hosted command and control platform for deploying and managing Hak5 devices remotely. It enables pentesters and red teams to provision gear, ship it to client sites, and access the full device arsenal from a web browser — without traversing firewalls or triggering detection systems.

## Compatible Devices

- WiFi Pineapple (Mark VII, Enterprise)
- LAN Turtle
- Packet Squirrel
- Screen Crab
- Shark Jack
- Key Croc

## Workflow

1. **Setup** — Provision Hak5 gear with Cloud C² in a few clicks
2. **Ship** — Send gear to client site; client only plugs it in
3. **Access** — Test their LAN and WiFi from anywhere via web browser

## Key Features

- **Web Terminal** — Drop into a root shell on any connected device from the browser
- **WiFi Pineapple Proxy** — Access full PineAP web interface as if physically on-site (Pro+)
- **Loot Exfiltration** — Transfer captured files and logs securely
- **AES Encryption** — Self-hosted, AES-secured; data stays on your own infrastructure
- **Multi-Device Management** — Manage multiple deployments simultaneously
- **Role-Based Access Control** — Audit logging and team permissions (Teams tier)
- **Plug-and-Provision** — USB drive setup for instant field deployment
