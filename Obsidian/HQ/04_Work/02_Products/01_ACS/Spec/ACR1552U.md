# ACR1552U
## USB NFC Reader IV — Recommended Replacement for ACR122U

### Overview
CCID & PC/SC compliant NFC reader with high-speed communication up to 848 Kbps and built-in SAM slot. Extensive card support including ISO 15693 (vicinity cards). USB firmware upgradability and extended APDU support (64 kbytes). Supports keyboard emulation mode. FIDO authentication compatible.

> ✅ Recommended replacement for the end-of-life [[ACR122U]].

### Key Features
- Three NFC modes: Card Reader/Writer, Card Emulation, Keyboard Emulation
- Reading distance up to 70 mm (vs 50 mm on ACR122U)
- High-speed communication up to 848 Kbps
- Extended APDU support (max 64 kbytes)
- Built-in SAM slot for key diversification and mutual authentication
- USB firmware upgradeability for post-deployment updates
- Anti-collision: one tag at a time
- FIDO authentication — compatible with PocketKey and YubiKey
- Short-circuit protection
- Keyboard emulation for broad software compatibility
