# ACR122U
## USB NFC Reader

### Overview
PC-linked contactless smart card reader/writer based on 13.56 MHz RFID. Compliant with ISO/IEC 18092 NFC. Supports MIFARE®, ISO 14443 Type A&B, and all four NFC tag types. CCID and PC/SC compliant — plug-and-play USB. Optional SAM slot for enhanced security.

### Key Features
- USB Full Speed Interface
- CCID & PC/SC compliant, plug-and-play
- Read/write up to 424 Kbps
- Supports all 4 NFC tag types (ISO/IEC 18092)
- Supports MIFARE Ultralight C, MIFARE Plus, DESFire EV1
- Three NFC modes: Reader/Writer, Card Emulation, Peer-to-Peer
- Built-in anti-collision (one tag at a time)
- Optional ISO 7816-3 SAM slot for enhanced security
- Available in module form for POS/vending machine integration
- User-controllable bi-color LED and buzzer
