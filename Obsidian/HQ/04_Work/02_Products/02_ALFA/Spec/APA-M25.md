# APA-M25
## 2.4GHz/5GHz Dual-Band High-Gain Indoor Panel Antenna

### Overview
High-gain dual-band indoor patch directional antenna. Same physical dimensions as APA-M25-6E but covers 2.4 + 5 GHz only. Tight 16° vertical beamwidth for focused directional coverage. Widely used as an upgrade antenna paired with AWUS036ACS.
