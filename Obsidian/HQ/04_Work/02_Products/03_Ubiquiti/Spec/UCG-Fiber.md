# UniFi Cloud Gateway Fiber
## Multi-WAN Cloud Gateway with IDS/IPS and NVR

### Overview
Compact desktop cloud gateway with dual 10G WAN (RJ45 + SFP+), 4× 2.5 GbE LAN, quad-core ARM CPU, and 5 Gbps IDS/IPS throughput. Supports NVMe SSD up to 2 TB for NVR recording.
