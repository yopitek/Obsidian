### 方案二：GX10 本地圖片伺服器（Gemini Web，無需 API Key）

**狀態：** ✅ 完成

**伺服器資訊：**
- 機器：ASUS Ascent GX10
- LAN IP：`192.168.1.228`
- Tailscale IP：`100.73.84.27`（⚠️ 本機 Tailscale 未加入同一 tailnet，LAN 連線即可）
- Gemini Image API：`http://192.168.1.228:8080/generate`
- ComfyUI：`http://192.168.1.228:8188`（可用）
- SSH User：`yopitek@192.168.1.228`

**API 使用方式：**
```bash
# 生成圖片（回傳 GX10 上的 image_path）
curl -X POST http://192.168.1.228:8080/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "your prompt here"}'
# 回傳：{"success":true,"image_path":"/home/yopitek/Downloads/image/gemini_XXXX.jpeg","error":null}

# 下載圖片（需 SSH 密碼）
sshpass -p 'PASSWORD' scp yopitek@192.168.1.228:/home/yopitek/Downloads/image/gemini_XXXX.jpeg ./output.png
```

**注意事項：**
- `/generate` 只回傳 JSON 路徑，不直接回傳圖片 bytes
- 無 HTTP 靜態檔案服務端點，需透過 SCP 下載
- SSH 支援密碼驗證（`publickey,password` 模式）
- 每張圖片生成約需 60–120 秒
