# Quality Assurance — 品質檢驗技能

## When to use

在**任何內容發布或儲存前**使用 QA Agent 進行檢查。包括：
- 電商商品上架前
- 社群貼文發布前
- 報告存入 Obsidian 前
- 影片發布前

## 檢查清單

### 必須檢查項目

1. **品牌一致性**
   - 品牌色是否正確
   - 品牌口吻是否一致
   - 標籤使用是否適當

2. **敏感詞合規**
   - 中國平台敏感詞替換
   - 競品描述中立性
   - 法律風險內容

3. **格式規範**
   - 圖片尺寸符合規格
   - 文字長度在範圍內
   - 標籤數量合理

4. **內容品質**
   - 開場吸引力
   - 視覺衝擊力
   - 資訊完整性
   - CTA 明確性

## 自動化檢查工具

```bash
# 檢查圖片尺寸
identify -format "%wx%h" image.jpg

# 檢查文字長度
wc -l content.txt

# 檢查敏感詞
grep -E "黑客 | 渗透测试|破解" content.txt
```

## 修正建議模板

```
問題類型：[品牌/敏感詞/格式/品質]
原始內容：[顯示問題內容]
建議修正：[具體修正方式]
優先級：[高/中/低]
```

## 通過標準

✅ **通過** — 所有檢查項目都符合標準
⚠️ **需要修正** — 有問題但可以自動修正或輕鬆修正
❌ **嚴重問題** — 需要大幅修改或重新製作
