# Agent Protocol: 10_resources

This directory is a resource hub. Follow these rules to ensure efficient discovery and usage.

## 🧭 Discovery Protocol
- **Check the Index First**: Before exploring subdirectories, always read `index.md` to identify the most relevant resource for your task.
- **Task Mapping**:
    - For **UI/Design/Multimedia**, prioritize folders in the `Content & Media Creation` category.
    - For **Engineering/Workflows**, prioritize `superpower` or other folders in `Coding & Agent Enhancement`.
    - For **Refining Text**, use the `Humanizer` suite in `Writing & Prompt Engineering`.

## 🛠️ Usage Rules
- **Skill Activation**: Most subdirectories contain a `SKILL.md`. Activate these using your platform's skill tool (e.g., `activate_skill`) to load specific instructions.
- **Pathing**: Use relative paths from the root of this directory when referencing resources.
- **Bilingual Support**: Note that many resources have Chinese (`zh`) and Traditional Chinese (`zh-TW`) versions. Choose the one that matches the user's language preference.

## 📁 Structure Guidelines
- Use `ls -F` to distinguish between standalone resource files and skill directories.
- If a resource is missing from `index.md`, please suggest an update to the user.
