## 結論：GX10 圖片生成最佳實踐

未來使用 GX10 生成產品圖時，**必須**加入 `reference_image` 參數：

```python
import json, subprocess, os

REF = "/home/yopitek/Downloads/image/ref_apa_m04.png"  # 先 scp 上傳到 GX10
payload = json.dumps({"prompt": "...", "reference_image": REF, "new_chat": True})
with open("/tmp/p.json","w") as f: f.write(payload)

r = subprocess.run(["curl","-sf","-X","POST","http://192.168.1.228:8080/generate",
    "-H","Content-Type: application/json","-d","@/tmp/p.json","--max-time","360"],
    capture_output=True, text=True, timeout=400)

remote_path = json.loads(r.stdout)["image_path"]
subprocess.run(["sshpass","-p","yopitek0401","scp",
    "-o","StrictHostKeyChecking=no",
    f"yopitek@192.168.1.228:{remote_path}", "./output.png"])
```
