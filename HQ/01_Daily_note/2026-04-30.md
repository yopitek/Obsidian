Windows 版 Claude Code 快速設定指南 (2026-04-30)

這份文件指導如何在 Windows PC 上安裝 Claude Code CLI，並連線至辦公室的 GX10 vLLM 引擎（透過 Tailscale 內網），同時**徹底解決「提示登入」的問題**。

---

## 🟢 1. 安裝環境 (Prerequisites)

1.  **安裝 Node.js**: 
    *   前往 [nodejs.org](https://nodejs.org/) 下載並安裝 **LTS 版本** (20.x 或更高)。
2.  **安裝 Claude Code CLI**:
    *   開啟 **PowerShell (以管理員身分)** 並執行：
    ```powershell
    npm install -g @anthropic-ai/claude-code
    ```

---

## 🛠 2. 繞過登入提示 (Bypass Login Prompt)

Claude Code 預設會導向瀏覽器登入。要讓它直接連線到 GX10 本地引擎，必須手動建立配置檔案。

### 步驟 A：建立 `.claude.json`
1.  按下 `Win + R`，輸入 `%USERPROFILE%` 並按回車。
2.  在該目錄下建立一個名為 `.claude.json` 的檔案（若已有則覆寫）。
3.  檔案內容如下：
```json
{
  "hasCompletedOnboarding": true,
  "primaryApiKey": "sk-ant-local-key",
  "customApiKeyResponses": {
    "approved": ["sk-ant-local-key"]
  }
}
```
*   **注意**：`primaryApiKey` 必須是以 `sk-ant-` 開頭的字串。

---

## 🌐 3. 設定內網連線變數 (Environment Variables)

你必須告訴 Claude Code 去哪裡連線。

### 方案 1：PowerShell 本次工作階段 (暫時)
在啟動 Claude 之前執行：
```powershell
$env:ANTHROPIC_BASE_URL="http://100.73.84.27:8000"
$env:ANTHROPIC_API_KEY="sk-ant-local-key"
$env:ANTHROPIC_DEFAULT_OPUS_MODEL="claude-3-5-sonnet-20241022"
$env:ANTHROPIC_DEFAULT_SONNET_MODEL="claude-3-5-sonnet-20241022"
$env:ANTHROPIC_DEFAULT_HAIKU_MODEL="claude-3-5-sonnet-20241022"
$env:CLAUDE_CODE_ATTRIBUTION_HEADER="0"
```

### 方案 2：永久設定 (推薦)
1.  在開始功能表搜尋 **「編輯系統環境變數」**。
2.  點擊 **「環境變數」** 按鈕。
3.  在 **「使用者變數」** 下方新增以下項目：

| 變數名稱 | 數值 |
| :--- | :--- |
| `ANTHROPIC_BASE_URL` | `http://100.73.84.27:8000` |
| `ANTHROPIC_API_KEY` | `sk-ant-local-key` |
| `ANTHROPIC_DEFAULT_OPUS_MODEL` | `claude-3-5-sonnet-20241022` |
| `ANTHROPIC_DEFAULT_SONNET_MODEL` | `claude-3-5-sonnet-20241022` |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL` | `claude-3-5-sonnet-20241022` |
| `CLAUDE_CODE_ATTRIBUTION_HEADER` | `0` |

---

## 🚀 4. 啟動與測試 (Launch)

### 4.1 連線檢查
在 PowerShell 中執行以下指令，確認能通到 GX10：
```powershell
curl.exe -X POST http://100.73.84.27:8000/v1/messages `
  -H "Content-Type: application/json" `
  -H "x-api-key: sk-ant-local-key" `
  -H "anthropic-version: 2023-06-01" `
  -d '{"model": "claude-3-5-sonnet-20241022", "messages": [{"role": "user", "content": "hi"}], "max_tokens": 10}'
```

### 4.2 啟動 Claude Code
設定完成後，直接輸入指令：
```powershell
claude
```
*(提示：若依然跳出登入視窗，請嘗試 `claude --bare`)*

## 💡 關鍵排錯 (Troubleshooting)

1.  **連線被拒絕 (Connection Refused)**：
    *   確認 GX10 上的 `start-stack-v2.sh` 已經啟動。
    *   確認 Windows 上的 Tailscale 已連線。
2.  **依然提示 Login**：
    *   檢查 `%USERPROFILE%\.claude.json` 檔案名稱是否有誤（開頭有一個點）。
    *   嘗試執行 `claude logout` 清除緩存。
3.  **亂碼問題**：
    *   Windows Terminal 建議使用 **UTF-8** 編碼。
