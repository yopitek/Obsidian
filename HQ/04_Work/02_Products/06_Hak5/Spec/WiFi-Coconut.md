# WiFi Coconut

> Multiple radios. One device. Capture all of 2.4 GHz WiFi simultaneously.

## Overview

The WiFi Coconut is a full-spectrum 2.4 GHz WiFi sniffer featuring 14 individual 802.11 radios — one for each channel. Unlike single-radio sniffers that channel-hop (missing 93% of traffic at any given time), the WiFi Coconut monitors all 14 channels at once from a single USB-C device.

## Key Features

- **Full Spectrum Capture** — All 14 × 2.4 GHz channels at once; zero missed traffic
- **PCAP Output** — Standard packet capture files for analysis
- **Kismet Integration** — Works natively with Kismet wireless monitor
- **Wireshark Integration** — Open PCAPs directly in Wireshark/tshark
- **Open Source Software** — [github.com/hak5/hak5-wifi-coconut](https://github.com/hak5/hak5-wifi-coconut)
- **USB-C Convenience** — Single cable connection
