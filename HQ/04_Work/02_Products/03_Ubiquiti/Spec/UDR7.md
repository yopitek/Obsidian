# UniFi Dream Router 7
## All-in-One WiFi 7 Router + NVR + UniFi Console

### Overview
Compact all-in-one UniFi console combining WiFi 7 tri-band AP, cloud gateway, and NVR. Includes 10G SFP+ uplink, 4× 2.5 GbE LAN with PoE, and a 64 GB microSD for camera recording.
