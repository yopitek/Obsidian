# Video Game Module for Flipper Zero
## Raspberry Pi RP2040 Gaming & Development Expansion

### Overview
Video streaming and motion-tracking module powered by a Raspberry Pi RP2040 microcontroller. Connects to Flipper Zero's GPIO header and adds TV video output, gyroscope/accelerometer input, and a dedicated USB-C port — enabling both gaming and standalone development. Compatible with existing Raspberry Pi Pico projects.

---

### Technical Specifications

| Parameter | Value |
|---|---|
| SKU | FLP-M-VGM |
| Microcontroller | Raspberry Pi RP2040 (dual ARM Cortex-M0+ @ 133 MHz) |
| Video Output | Composite video — TV connection |
| Sensors | Built-in gyroscope + accelerometer (IMU) |
| GPIO Port | 14-pin expansion connector for external components |
| USB Interface | USB-C (direct to RP2040 — independent of Flipper Zero) |
| Firmware | Open-source; schematics published |
| Pico Compatibility | Compatible with Raspberry Pi Pico firmware |
| Standalone Mode | ✅ Functions without Flipper Zero connected |
